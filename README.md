# Project-partner-finder
The Project Partner / Group Finder system is a web-based application designed to assist students in finding suitable project partners and forming project groups based on common interests, skills, and project preferences. The system aims to reduce the manual effort involved in group formation and improve collaboration efficiency. It provides features such as user profile creation, partner search, group creation, communication, and task coordination, thereby supporting students throughout the project lifecycle.

---

## Technologies Used

- MongoDB  
- Express.js  
- React.js  
- Node.js (MERN Stack)

---

## Key Features

- User authentication and secure login  
- User profile creation with skills and interests  
- Project partner and group matching  
- Group creation and management  
- Real-time communication between group members  
- Task coordination within groups  
- Responsive and user-friendly interface
